Question #1:
What is DOM and what is the purpose of it?

Ans:
DOM (Document Object Model) is an interface that represents how your HTML documents are read by the browser. It allows javascript to manipulate your HTML document. You can do infinite number of functional things with your code. So, the purpose of DOM is to make your website more functional.

Question #2:
How to select element from DOM?

Ans:
You can select an element from DOM in 5 ways
1. getElementsByTagName
2. getElementsByClassName
3. getElementById
4. querySelectorAll
5. querySelector

Question #3:
What is Event Bubble?

Ans:
Event Bubble is part of DOM concept. It occurs when an event happens and that event goes to it's parent element.

Question #4:
What is Callback Function?

Ans:
A callback function is a function passed into another function as an argument after that it invokes the outer function to complete some new command.
